Features

Across data sources

It provides a uniform API for accessing stuff from different kinds of databases, protocols, and 3rd party APIs. That means you write the same code to get and store your business objects, whether its data source live in database, MongoDB, Redis, whatever else.

Intuitive queries

Nice and easy query builder that helps your create your model layer after a few minutes of documentation reading.

ORM concept

Contains everything that is expected from usuall ORM library, so you can focus only on entities and the operations above them.

Fully extendable

We do not tell you strictly how to use it. There are some built-in queries, mappers and framework extensions, but if you need something more, just write your own extension based on official API.

Minimal requirements

You don't need any framework, extensions or third part libraries. Everything you need is PHP 5.4+ and some nice application for the use of this tool.

Opensource

Everything is licensed under the MIT license so you can use it in your applications arbitrarily.
